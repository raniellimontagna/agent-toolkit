# Security Skills Attribution

These first-party Agent Toolkit skills are informed by public defensive
security and reconnaissance patterns from
[`uphiago/recon-skills`](https://github.com/uphiago/recon-skills), licensed
under the MIT License. They are rewritten and narrowed for authorized,
non-destructive application and code security reviews; the upstream agent
instructions, operational infrastructure assumptions, scripts, and offensive
workflows are not included.
